=====DIRTY FLAG PATTERN=====

Both programs are written in Java.

To compile:
- Open files in an IDE such as Eclipse or IntelliJ and run.
OR
- From console, navigate to directory containing both test and project file(s) and type "javac *.java" to compile all .java files.
- Next type "java <projectname>" where <projectname> is the name of the test file containing our main method.
